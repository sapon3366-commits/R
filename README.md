# 登入驗證頁面

一個現代化、美觀的登入驗證頁面，使用純 HTML、CSS 和 JavaScript 開發。

## 功能特色

- ✨ 現代化的 UI 設計
- 🔐 表單驗證（電子郵件格式、密碼長度）
- 👁️ 顯示/隱藏密碼功能
- 💾 記住我功能（使用 localStorage）
- 📱 響應式設計，支援手機和平板
- ⚡ 流暢的動畫效果
- 🎨 漸層背景和卡片式設計

## 檔案結構

```
.
├── index.html      # 主要 HTML 檔案
├── styles.css      # 樣式表
├── script.js       # JavaScript 邏輯
├── .gitignore      # Git 忽略檔案
└── README.md       # 說明文件
```

## 使用方法

1. 直接在瀏覽器中開啟 `index.html` 檔案
2. 或使用本地伺服器：
   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 使用 Node.js (需要安裝 http-server)
   npx http-server
   ```
3. 在瀏覽器中訪問 `http://localhost:8000`

## 測試帳號

- **電子郵件**: `test@example.com`
- **密碼**: `password123`
